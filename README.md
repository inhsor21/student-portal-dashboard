# Student Portal Dashboard

This is a static student portal dashboard built using HTML and CSS. The aim was to create something that looks like an actual university dashboard instead of a basic practice project.

## About the Project

The dashboard includes a sidebar, a top navigation bar, and a main content area with sections like stats, timetable, announcements, and assignments. The layout is designed to feel like a real student portal used in universities.

For this project, I also wanted to try out a more prompting-based workflow using Claude instead of building everything completely from scratch. I focused on clearly defining what I wanted (layout, sections, design style, etc.) and then iterated on the output to improve the structure and UI.

You could say this was me experimenting with a bit of “vibe coding” — guiding the build through intent and refining it along the way.

## Features

* Sidebar navigation (Dashboard, Subjects, Assignments, Attendance, Profile, Settings)
* Top navigation bar with profile and notification UI
* Dashboard sections including:

  * Welcome banner
  * Academic statistics (attendance, GPA, tasks, credits)
  * Timetable / upcoming classes
  * Announcements
  * Assignments table
* Clean and modern UI
* Responsive layout
